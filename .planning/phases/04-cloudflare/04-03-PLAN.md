```python
def health_check():
    """Health check endpoint for monitoring."""
    from app import db
    from flask import current_app

    try:
        # Check database connection
        db.session.execute('SELECT 1')
        db_status = 'healthy'
    except Exception as e:
        db_status = f'unhealthy: {str(e)}'

    # Check application status
    app_status = 'healthy' if current_app else 'unhealthy'

    return {
        'status': 'ok' if db_status == 'healthy' and app_status == 'healthy' else 'degraded',
        'timestamp': datetime.utcnow().isoformat(),
        'services': {
            'database': db_status,
            'application': app_status
        },
        'version': '1.0'
    }
```

<action>
创建 DNS 自动化设置脚本，使用 cloudflared 命令创建 CNAME 记录。

脚本功能：
1. 检查必需环境变量：`CLOUDFLARE_TUNNEL_ID`, `DOMAIN`, `BLOG_SUBDOMAIN`, `STATUS_SUBDOMAIN`
2. 创建博客子域名 CNAME 记录：
   ```bash
   cloudflared tunnel route dns "${CLOUDFLARE_TUNNEL_ID}" "${BLOG_SUBDOMAIN}.${DOMAIN}"
   ```
3. 创建状态子域名 CNAME 记录：
   ```bash
   cloudflared tunnel route dns "${CLOUDFLARE_TUNNEL_ID}" "${STATUS_SUBDOMAIN}.${DOMAIN}"
   ```
4. 验证 DNS 记录创建：
   ```bash
   cloudflared tunnel info "${CLOUDFLARE_TUNNEL_ID}"
   ```
5. 提供 DNS 传播等待提示（通常需要几分钟到几小时）

脚本要求：
- 清晰的输出，显示每个步骤的结果
- 错误处理：如果命令失败，提供修复建议

- 环境变量验证：确保所有必需变量已设置
